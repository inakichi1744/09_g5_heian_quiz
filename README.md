# 平安時代クイズ（小5・6月）

中学受験サポート用のクイズアプリです。小5・6月の演習から、全19問を出題します。

## 出題内容
- 平将門の乱・後三年合戦
- 国風文化（かな文字・寝殿造・平等院鳳凰堂）
- 空海と真言宗、白河上皇の院政
- 平清盛・日宋貿易（太政大臣・大輪田泊・厳島神社）
- 保元の乱・平治の乱
